## Design Description

- **Style**: Elegant minimal cyberpunk aesthetic
- **Color Palette**:
  - Primary Dark: #0a0a0a, #111111, #1a1a1a
  - Neon Cyan: #00ffff, #64ffda, #00e5ff
  - Text: #ffffff, #b0b0b0, #888888
- **Typography**:
  - Headers: 'JetBrains Mono' (monospace for tech feel)
  - Body: 'Inter' (clean, modern sans-serif)
- **Moodboard Keywords**: Cyberpunk, minimal, neon, terminal, matrix, sleek, professional, futuristic

## Files to Create/Modify:

1. **index.html** - Main structure with all sections
2. **style.css** - Dark theme styling with neon accents and animations
3. **script.js** - Interactive animations (terminal typing, glitch effects)

## Sections Implementation:

1. **Hero Section**: Name, tagline, animated terminal typing effect
2. **About Section**: Brief description with avatar placeholder
3. **Projects Grid**: 3 project cards with hover effects
4. **Skills Section**: Animated tag cloud with neon glow
5. **Contact Section**: Email and social media icons
6. **Navigation**: Smooth scrolling, mobile hamburger menu

## Animation Ideas:

1. **Terminal Typing**: Typewriter effect for hero tagline
2. **Glitch Effect**: Text distortion on hover
3. **Particle Background**: Subtle moving dots/lines

## Responsive Design:

- Mobile-first approach
- Breakpoints: 768px, 1024px, 1200px
- Flexible grid system
- Touch-friendly navigation
